# Simple Multi-Page Website

This is a simple multi-page website built using HTML and CSS.

## Pages Included
- Home
- About
- Contact

## Features
- Common navigation bar across all pages
- Single CSS file reused for consistent styling
- Relative paths used for navigation
- Clean and simple layout

## Technologies Used
- HTML
- CSS
